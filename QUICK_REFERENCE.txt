⚡ COMMON COMMANDS
================================================================================

   pip install -r requirements.txt

Run Streamlit app:
   streamlit run streamlit_app.py

Run on different port:
   streamlit run streamlit_app.py --server.port=8502

Run Flask backend (optional):
   python flask_app.py

Check Python version:
   python --version

================================================================================
❌ ERROR FIXES
================================================================================

ERROR: "Cannot connect to API"
FIX: You don't need the API! Streamlit has built-in data.
     Just run: streamlit run streamlit_app.py

ERROR: "streamlit: command not found"
FIX: pip install streamlit

ERROR: "ModuleNotFoundError"
FIX: pip install -r requirements.txt

ERROR: "Port 8501 already in use"
FIX: streamlit run streamlit_app.py --server.port=8502

ERROR: "No such file or directory"
FIX: Make sure streamlit_app.py is in your current folder
     Check with: dir (Windows) or ls (Mac/Linux)
